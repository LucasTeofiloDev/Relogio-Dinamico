# Relôgio Dinâmico
Projeto desenvolvido para praticar a manipulação de elementos e estilos em tempo real com JavaScript.

## Funcionalidades
O sistema identifica a hora atual do sistema, exibe a saudação correspondente e altera dinamicamente a imagem e a cor de fundo da página de acordo com o período (manhã, tarde ou noite).

## Tecnologias
JavaScript (ES6+)

HTML5

CSS3

## Conceitos Aplicados
A implementação utiliza o objeto Date para capturar dados temporais do sistema, manipulação de propriedades de estilo do CSS através do DOM e atualização dinâmica de atributos de imagem baseada em condições lógicas.
